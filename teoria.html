Teoria JavaScript

1) Kiedy powstał JavaScript?
W grudniu 1995 roku.

2) Dlaczego jest to język skryptowy?
JavaScript jest interpretowany na bieżąco, a nie kompilowany przed wykonaniem.

3) Gdzie jest wykonywany JS?
Na kliencie (przeglądarki internetowe).

4) Dlaczego JS to bezpieczne narzędzie?
Nie ma bezpośredniego dostępu do systemu plików użytkownika.

5) Czy jest to język obiektowy? Jeśli tak, to dlaczego?
Tak, ponieważ obsługuje obiekty i ich metody.

6) Wymień zalety JS:
Zaleta 1: Uniwersalność.
Zaleta 2: Łatwa integracja z HTML.
Zaleta 3: Obsługa interaktywnych elementów.

7) Framework to:
Biblioteka wspomagająca tworzenie aplikacji.

8) Do aplikacji mobilnych można użyć Framework:
React Native.

9) Aplikacje responsywne to:
Strony dostosowujące się do różnych urządzeń.

10) Kod:
Wyświetla przycisk otwierający nowe okno z funkcją `WinOpen`.

11) Kod: window.open(...)
Otwiera nowe okno przeglądarki o określonych parametrach.

12) Kod: window.close()
Zamyka aktualnie otwarte okno przeglądarki.
